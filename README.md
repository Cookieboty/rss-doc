# RssTabs

基于Figma设计的RSS阅读器应用，使用Next.js和Tailwind CSS构建。

## 功能

- 现代化UI设计
- 响应式布局
- RSS订阅管理
- 文章阅读和收藏

## 技术栈

- Next.js
- React
- TypeScript
- Tailwind CSS
- Heroicons

## 开始使用

1. 克隆仓库
```bash
git clone https://github.com/yourusername/rsstabs.git
cd rsstabs
```

2. 安装依赖
```bash
npm install
```

3. 运行开发服务器
```bash
npm run dev
```

4. 在浏览器中打开 [http://localhost:3000](http://localhost:3000)

## 构建生产版本

```bash
npm run build
```
